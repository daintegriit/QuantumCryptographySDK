# QuantumShield

An AI-enhanced post-quantum cryptography SDK designed to secure the future.

## Tech Stack

- **Rust**: High-performance encryption core
- **Python (FastAPI)**: API wrapper
- **PyTorch**: Neural cryptanalysis
- **NumPy / matplotlib**: Benchmark analysis
- **Docker**: Containerized deployment
- **AWS Lambda / VPC**: Scalable cloud deployment
